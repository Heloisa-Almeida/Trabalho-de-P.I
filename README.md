# 💉VaccinMonitorate
## 📖 Sobre o Projeto
Projeto desenvolvido em grupo com a finalidade de diminuir as perdas de vacinas por falta de monitoramento eficaz da temperatura.

## ⚠️ Problema Atual
Atualmente, existem muitos casos de perdas de vacinas devido à falta de monitoramento da temperatura em seu transporte, causando atrasos, risco à imunização, perda dos componentes ativos da vacina e principalmente prejuízos financeiros.
